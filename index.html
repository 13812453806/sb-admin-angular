SB Admin v2.0 rewritten in AngularJS

This project is a port of the famous Free Admin Bootstrap Theme SB Admin v2.0 to Angular Theme. Find out more Free Angular Themes at StartAngular.com.

Installation

  
    cd project-directory
    npm install
    bower install
    grunt serve:dist --force

If you want to run the development version and skip throught the minification process ,

grunt serve --force

Features

  • All components made into directives for ease of use.
  • Lazy loading using ocLazyLoad
  • Responsive and reactive charts using Angular-charts
  • Files are minified for faster loading with grunt

How to add more stuff?

Its very simple to add new files or features to this theme. Follow these simple steps,

  • Go to the app.js file and add a state, give it a view and controller. Lazy load it if you feel it is needed. Check out this doc if you have trouble with lazy loading.
  • Now add the view in the Views folder and the controller in the scripts/controllers folder.
